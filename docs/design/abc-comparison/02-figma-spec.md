### 1.3 关键组件规格

#### Component: `GradeCell`（评级单元格）

```
┌─────────────────────────┐
│  AA · AB ↓              │   padding: 2px 6px
└─────────────────────────┘   radius: 4px
                              hover bg: #f0f5ff

字符规则：
- 本店值：14px / 600 / #262626
- 分隔符 "·"：12px / 400 / #d9d9d9 / margin: 0 2px
- 组内值：14px / 400 / #595959
- 箭头：12px / 700 / margin-left: 2px
```

**5 种变体（Variant）：**

| Variant | 本店 | 分隔 | 组内 | 箭头 | 行底色 |
|---|---|---|---|---|---|
| `equal` | AA | · | AA | = #bfbfbf | #ffffff |
| `up-1` | AA | · | AB | ↑ #52c41a | #ffffff |
| `up-2` | AA | · | BC | ↑↑ #389e0d | #ffffff |
| `down-1` | AB | · | AA | ↓ #fa8c16 | #fffbe6 |
| `down-2` | BC | · | AA | ↓↓ #f5222d | #fff1f0 |

#### Component: `SequenceBadge`（序号徽标）

```
○ 1     圆形 22 × 22
        background: #1890ff
        color: #fff
        font: 12px / 600
        前 3 名 #1890ff，4 名后 #bfbfbf
```

#### Component: `ItemCell`（序号+物料合并列）

```
┌────────────────────────────────────┐
│  ① │ [新] 梦龙榛果巧克力派           │
└────────────────────────────────────┘
gap: 8px
名称最大宽度: 180px，超出省略号
"新" 角标：bg #ff4d4f / color #fff / font 10px / radius 2px / padding 0 4px
```

#### Component: `QuantityMerged`（建议→目标销量）

```
┌──────────────────────────────────┐
│  35  →  [−] 35 [+]                │
└──────────────────────────────────┘
建议值：14px #1890ff 可点击（点击后填入到目标销量框）
箭头 →：12px #d9d9d9
目标销量：步进器 24px 高，输入框宽 36px
```

#### Component: `PromoCollapsed`（营销活动折叠）

```
[加价购] +3       仅显示首个 + 数量
```

#### Component: `Drawer/Bottom`（底部抽屉）

```
- 蒙层：rgba(0,0,0,.4)
- 抽屉宽：600px (max-width 90%)
- 高：自适应内容，max-height 70%
- 圆角：16px 16px 0 0
- 内边距：24px
```

#### Component: `Drawer/Right`（右侧抽屉，用于列设置）

```
- 宽：380px
- 高：100%
- 圆角：16px 0 0 16px
- 内边距：24px
```

```

### 提示文案规则（V1.0 仅文案，不做 CTA）

| 偏离 | 文案 |
|---|---|
| `=` | 本店表现与同类型门店一致 |
| `↑` 或 `↑↑` | 本店在【XX组】中评级领先 N 档，可作为标杆经验分享 |
| `↓` 或 `↓↓` | 本店在【XX组】中评级偏低 N 档，同类型门店该产品多为 XX 级，建议加大订货量 |

│ │ ○  │A类店           │店等级│  180  │  [开]   │  2   ││
│ │ ○  │华南区·现烤组   │区域+ │  120  │  [开]   │  3   ││
│ │    │                │品类  │       │         │      ││
│ └────┴───────────────┴──────┴───────┴────────┴──────┘│
│                                                         │
│ 💡 说明：当门店属于多个分析组时，系统按"默认对标组"      │
│    展示组内 ABC 对标。门店级未配置时，按"分析组优先级"   │
│    自动选择最小粒度组。                                  │
│                                                         │
│                              [取消]    [保存]            │
└────────────────────────────────────────────────────────┘
```

### 字段说明

| 字段 | 说明 | 是否新增 |
|---|---|---|
| 分析组名称 | 已有 | 否 |
| 分组维度 | 已有 | 否 |
| 包含门店 | 已有，本页面通过"所属分析组"反查 | 否 |
| **是否可作为对标组** | 开关，仅"开"的组会出现在前端下拉 | **新增** |
| **优先级** | 1—N 数字，门店未指定默认组时按此自动选择 | **新增** |
| **门店级默认对标组** | 单选，默认配置 | **新增** |

### 智能默认值规则（兜底）

如果某门店未在此页面配置默认对标组，按以下顺序自动选择：

1. 在该门店所属、且"可作为对标组"开启的组中，取**优先级数字最小**的组
2. 如果优先级相同，取**包含门店数量最少**的（粒度最细）
3. 如果还是相同，取**最早创建**的组
4. 兜底：所有门店

## 6. 在 Figma 中如何快速搭建

### Step 1：导入颜色 Token
- 在 Figma 中创建 Variables → Color Mode → 把上面表格的颜色逐条录入
- 命名空间：`primary/500`、`success/500` 等，与本文档一致

### Step 2：创建文字样式
- 选中文本 → 创建 Text Style → 命名为 `body/md`、`kpi-num` 等

### Step 3：组装核心组件
按"1.3 关键组件规格"创建：
- `GradeCell`（5 种 Variant）
- `SequenceBadge`（2 种 Variant：active / disabled）
- `ItemCell`、`QuantityMerged`、`PromoCollapsed`
- `Drawer/Bottom`、`Drawer/Right`

### Step 4：搭建 Frame
按"2—5 节"逐个 Frame 搭建。

### Step 5：连接交互（Prototype）
- 评级单元格 → 点击 → 弹出 `Drawer/Bottom`
- 评级列头 → 点击 → 弹出下拉
- ⚙ 图标 → 点击 → 弹出 `Drawer/Right`
